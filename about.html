GPCR–Human Ligand Interactome (Boltz-2)

Project Overview

The GPCR–Human Ligand Interactome is an interactive resource for exploring predicted structural complexes between human G protein–coupled receptors (GPCRs) and endogenous ligands.

All structures were generated using the Boltz-2 deep-learning framework, producing receptor–ligand complexes together with confidence metrics (pLDDT, iPLDDT, ipTM, iPDE) and predicted affinities.

What This Site Provides

  • Searchable table of GPCR–ligand pairs
  • Interactive 3D visualization of predicted complexes
  • Residue-level inspection with neighborhood highlighting
  • Confidence metrics for every model
  • Direct download of structural files (CIF)

Data and Methods

Structural models were generated with Boltz-2 using curated human GPCR sequences and a library of endogenous ligands. Each complex is accompanied by model confidence scores and predicted binding affinity.

All structure files are hosted on Zenodo and are versioned and citable. The web interface is implemented using GitHub Pages and 3Dmol.js.
